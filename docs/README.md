# Documentation

This directory is supposed to hold all the necessary documentation related to the project.

Specifically the following 2 documents are mandatory and must be provided together with the source code:
- **Technical Report** defining the Business Problem and a detailed description of the approach used as well as the results obtained by the project.
- **Operational Manual** defining the necessary steps for the deployment of teh project in a target environment and the expected life-cycle of it.

The document above are already included and provided as *templates* to be filled in.

*It is possible to just link the above documents in this README if they are hosted onto another Mondi system or service.*
